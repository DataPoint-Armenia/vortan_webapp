# vortan_webapp

Webapp for vortan spellchecker.

## Install

```
cd frontend
npm install
```

## Run

Frontent
```
cd frontend
npm run start
```

Backend
See [vortan_api](https://github.com/datapoint-armenia/vortan_api)

## Deploy

```
cd frontend
npm run deploy
```
